# Expense Tracker

## 📖 Project Description

Expense Tracker is a web-based application that helps users track and manage daily expenses efficiently. It allows users to record expenses, organize them into categories, and monitor spending habits through a simple and user-friendly interface.

## Technologies Used

* HTML
* CSS
* JavaScript
* Local Storage

## Features

* Add Expenses
* Delete Expenses
* Category Management
* Expense Summary
* Data Persistence using Local Storage
* Responsive User Interface

## Future Scope

* User Authentication

* AI-based Spending Analysis
* Export Reports (PDF/Excel)
* Budget Planning and Alerts

##  Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/4rnav-codes/Expense-Tracker.git
   ```
2. Open the project folder.
3. Run `index.html` in your browser.
